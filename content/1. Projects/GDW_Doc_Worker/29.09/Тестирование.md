## Тест-кейс 1. Папка, del-all-by

1. Выбрать папку с docx файлами;
2. Указать необходимый уровень или диапазон заголовков (в нашем случае диапозон 1-2);
3. Выбрать папку, в которую будут сохранены порезанные фрагменты документа;
4. Выбрать режим работы (в нашем случае del-all-by).


**Ожидаемый результат:**
В программе есть отдельное окошко логов, оно должно в реальном времени отображать происходящие процессы.

1. Скрипт считывает docx документы из указанной директории;
2. Скрипт парсит документ, разбивая его на 4 сегмента по заголовкам (всего должно выйти 4 файла, с заголовками 1-1, 1-2, 2-1, 2-2);
3. Скрипт сохраняет каждый сегмент в отдельный docx файл в указанной директории;
4. Скрипт уведомляет пользователя об окончании процесса и завершает работу.

**Фактический результат:**
Это первый скрипт, в котором корректно работает окошко с логами. Ура.

1. Скрипт считал docx файлы из указанной директории;
2. Скрипт спарсил документ, разбив его на сегменты 1-1, 1-2, 2-1, 2-2 (всё верно, скрипт не вышел за предел установленного диапазона и не создал отдельный файл для сегмента с заголовком 3-1);
3. Скрипт сохранил каждый сегмент в отдельный docx файл в указанной директории;
4. Скрипт уведомил пользователя об окончании процесса, вывел общие результаты работы в окошко с логами и корректно завершил работу.

Это победа! И логи, и завершение работы. Всё работает.
