# Измненения
## Плагин `GDW_Doc_Worker.py`

Убрал два неиспользуемых импорта: `sys` и `filedialog` из `tkinter`.

Было:
```python
import os
import sys # Его убрал
import importlib.util
import tkinter as tk
from tkinter import ttk, messagebox, filedialog # И вот его тоже
```

Стало:
```python
import os
import importlib.util
import tkinter as tk
from tkinter import ttk, messagebox
```

# Плагин `GDW_CleanMD`

В `def _stop(self)` добавлено принудительное завершение задач в `ThreadPoolExecutor`, остановка очереди и обновление интерфейса:
```python
# Принудительное завершение задач в ThreadPoolExecutor
if hasattr(self, 'executor'):
	self.executor.shutdown(wait=False)
	
    # Остановка очереди и обновление интерфейса
    self._queue.put(("done"))
    self.btn_start.config(state="normal")
    self.btn_stop.config(state="disabled")
```

Разница версий:
![[CleanShot 2025-09-29 at 17.00.53@2x.png | 600]]

Теперь скрипт завершает свою работу, и логи кое-как заработали. Всё ещё криво, но уже что-то.

В первый запуск логов нет, работа не завершается автоматически. Но кнопка "Остановить" работает.
После второго запуска работают логи (наполовину) и скрипт автоматически завершает работу.
