# Общие баги

## Баг №1
Ни в одном плагине в поле для пути до файла/директории нельзя вставить путь из буфера обмена. Путь можно написать вручную, можно выбрать через проводник, но вставка из буфера обмена невозможна.

# Плагин `GDW_DocSplitter`

## Баг №1
При нажатии на кнопку "Обзор" для выбора файла программа вылетает, но если выбрать режим выбора папки, а не файла - всё окей.
Если в режиме "Файл" прописать путь до файла вручную - всё сработает корректно. Вылет только при нажатии на кнопку "Обзор" в режиме "Файл".

# Плагин `GDW_Convert`

## Баг №1
При конвертации docx в формат "smart" ничего не происходит, файл не создаётся. С этим форматом какие-то проблемы. Возможно, у меня в системе не хватает какой-то зависимости для работы с этим форматом, не знаю.

## Баг №2
Скрипт не может завершить работу. Ни автоматически, ни при помощи кнопки "Стоп". Помогает только полный перезапуск программы.
