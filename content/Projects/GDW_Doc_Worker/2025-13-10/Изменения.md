Сделал так, чтобы картинки короче из документа в отдельную папку складировались. Но пока с нюансом. Забираются вообще все картинки. Нет какой-то выборки по заголовкам, в тупую все картинки забираются.

Написана новая функция `_extract_images`

```python
# === Вытащить картинки из документа === #
    def _extract_images(self, docx_path, out_dir):
        # Создаём папку для картинок, если её нет
        os.makedirs(out_dir, exist_ok=True)
        with zipfile.ZipFile(docx_path, 'r') as z:
            for name in z.namelist():
                if name.startswith('word/media/'):
                    img_name = os.path.basename(name)
                    img_path = os.path.join(out_dir, img_name)
                    base, ext = os.path.splitext(img_path)
                    i = 1
                    while os.path.exists(img_path):
                        img_path = f"{base}_{i}{ext}"
                        i += 1
                    with z.open(name) as src, open(img_path, 'wb') as dst:
                        shutil.copyfileobj(src, dst)
```
