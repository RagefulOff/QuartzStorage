```python
if self.zip_var.get():
    zip_path = os.path.join(os.path.dirname(self.converted_root), "Convert.zip")
    try:
        with zipfile.ZipFile(zip_path, "w", compression=zipfile.ZIP_DEFLATED) as zf:

                for file in files:
                    full_path = os.path.join(root, file)

        self._log(f"Ошибка архивирования: {e}")
```

# Плагин `GDW_CleanMD`

Прежде чем добавить архивацию, нужно сделать так, чтобы плагин не менял существующий файл, а как `GDW_Convert` кидал изменённые файлы в отдельную директорию.

>[!attention] Внимание
>При таком раскладе функция бэкапа больше не имеет смысла, поэтому будет удалена.

## Функция `_start`

Добавлено создание директории для хранения изменённых файлов.

```python
# Создание директории для сохранения обработанных файлов
self.output_dir = os.path.join(root, "output_md")
os.makedirs(self.output_dir, exist_ok=True)
```

---
## Функция `_process_file`

Удалена строчка записи изменений в файл из блока сохранения изменений:

```python
with open(path, "w", encoding="utf-8") as f:
    f.write(text)
```


Добавлен блок кода с созданием изменённого файла в отдельно директории:

```python
# Сохранение обработанного файла в новой директории
rel_path = os.path.relpath(path, root)
new_path = os.path.join(self.output_dir, rel_path)
os.makedirs(os.path.dirname(new_path), exist_ok=True)
                
with open(new_path, "w", encoding="utf-8") as f:
    f.write(text)
```

## Функция `_process_file`

Был написан блок кода для архивирования результатов, так же вместе с этим блоком был слегка изменён блок финального логирования, туда просто были добавлены логи по архивации:

```python
# Архивирование
if self.zip_var.get():
    zip_path = os.path.join(os.path.dirname(self.output_dir), "output_md.zip")
    try:
        with zipfile.ZipFile(zip_path, "w", compression=zipfile.ZIP_DEFLATED) as zf:
            for root, _, files in os.walk(self.output_dir):
                for file in files:
                    full_path = os.path.join(root, file)
                    arcname = os.path.relpath(full_path, os.path.dirname(self.output_dir))
                    zf.write(full_path, arcname=arcname)
        zip_log = f"ZIP-архив создан: {arcname}"
    except Exception as e:
        zip_log = f"Ошибка архивирования: {e}"


# Логирование результата
msg = f"Применено: {(", ".join(changes) if changes else "без изменений")}\n\n  {zip_log}"
```

## Функция `_run_worker`

В самом начале функции объявлен список `created_files` для сбора всех созданных скриптом файлов:

```python
created_files = []
```


В список добавляется каждый созданный файл (или директория):

```python
self.saved += 1
created_files.append(out) # <= это вот здеся добавилось
self._log(f" ✓ {fn}.docx")
```


Ну и сама функция архивации:

```python
# Архивация
if self.zip_var.get():
    zip_path = os.path.join(dst, "result.zip")
    try:
        with zipfile.ZipFile(zip_path, "w", compression=zipfile.ZIP_DEFLATED) as zf:
            for root, _, files in os.walk(dst):
                for file in created_files:
                    full_path = os.path.join(root, file)
                    arcname = os.path.relpath(full_path, dst)
                    zf.write(full_path, arcname=arcname)
        self._log(f"\nZIP-архив создан: {zip_path}")
    except Exception as e:
        self._log(f"\nОшибка при создании ZIP-архива: {e}")
```
