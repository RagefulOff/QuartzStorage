### Попытка №1
#### Функция `_bind_mac_shortcuts`

Пытался написать привязку к русским символам напрямую, для обработки сочетаний  клавиш `cmd+С`, `cmd+М`, `cmd+Ч` к необходимым действиям (копировать, вставить, вырезать):

```python
# Починка буфера обмена на Mac
def _bind_mac_shortcuts(self, entry):
    entry.bind("<Command-м>", lambda event: entry.event_generate("<<Paste>>"))
    
    entry.bind("<Command-с>", lambda event: entry.event_generate("<<Copy>>"))
    
    entry.bind("<Command-ч>", lambda event: entry.event_generate("<<Cut>>"))
```

#### Функция `create_frame`

В функции был изменён `ttk.Entry`  для ввода пути до файла/каталога. Вернее само поле изменено не было, оно просто передалось в отдельную переменную для работы биндов на мак. Ну и вызов функции биндов соответственно.

Было:
```python
ttk.Entry(frame, textvariable=self.src_dir)\
    .grid(row=0, column=1, columnspan=3, sticky="ew", padx=5)
```

Стало:
```python
entry = ttk.Entry(frame, textvariable=self.src_dir)
entry.grid(row=0, column=1, columnspan=3, sticky="ew", padx=5)
self._bind_mac_shortcuts(entry)
```

Tkinter вообще не обрабатывает кириллицу как клавишу. Выходит ошибка `Не удалось загрузить GDW_Convert.py: bad event type or keysym "м"`

### Попытка №2

Так как напрямую ткинтер не хочет работать с кириллицей, было решено пойти в обход.

#### Функция `_bind_mac_shortcuts`

Функция обрабатывает не конкретную привязку, а cmd + что угодно ещё, а если это "что угодно" одна из нужных клавиш, будет вызывать `self._handle_command_key`

```python
def _bind_mac_shortcuts(self, entry):
    entry.bind("<Command-KeyPress>", self._handle_command_key)
```

#### Функция `_handle_command_key`

Функция проверяет, была ли нажата нужная кнопка, и если да - вызывает нужное событие.

```python
def _handle_command_key(self, event):
    if event.keysym.lower() is "м":
        event.widget.event_generate("<<Paste>>")
    elif event.keysym.lower() is "с":
        event.widget.event_generate("<<Copy>>")
    elif event.keysym.lower() is "ч":
        event.widget.event_generate("<<Cut>>")
```

Ну и вызов функции:
```python
entry = ttk.Entry(frame, textvariable=self.src_dir)
entry.grid(row=0, column=1, columnspan=3, sticky="ew", padx=5)
self._bind_mac_shortcuts(entry)
```

Тоже не сработало.
