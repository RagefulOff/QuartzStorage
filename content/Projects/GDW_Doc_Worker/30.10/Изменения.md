## Функция `_start`

В функцию `_start` добавлен блок с сбросом состояния программы

Было:
```python
def _start(self):
        src = self.src_dir.get().strip()
        if not src or not os.path.isdir(src):
            return messagebox.showwarning("Ошибка",
                                          "Укажите корректную папку")
```

Стало:
```python
def _start(self):
        src = self.src_dir.get().strip()
        if not src or not os.path.isdir(src):
            return messagebox.showwarning("Ошибка",
                                          "Укажите корректную папку")

        # Сброс состояния перед новым запуском
        self._stop_requested = False
        self._queue          = queue.Queue()  # Очистка очереди
        self.success         = 0
        self.errors          = 0
        self.start_time      = time.time()
```
