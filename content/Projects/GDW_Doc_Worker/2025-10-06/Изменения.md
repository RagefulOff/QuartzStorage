# Батники

## Батник для установки зависимостей

В проекте был создан файл `requirements.txt` с следующими зависимостями:

```
docx
python-docx
```

---

Далее был написан `.bat` файл, который сам устанавливает все зависимости, прописанные в файле `requirements.txt`:

```bash
@echo on
pip install -r "requirements.txt"
pause
```

---
## Батник для запуска скрипта

Вот `.bat` файл, запускающий скрипт:

```bash
@echo off
start "" pythonw Graf_Doc_Worker.py
exit
```

`start ""` - отвязывает батник от выполнения питоновского скрипта, чтобы окно консоли батника не висело всё время выполнения программы, а закрывалось сразу же после запуска.

`pythonw` - запуск исключительно графического интерфейса программы. Если запустить скрипт через `python` - вместе с самой программой откроется пустое консольное окно питона.

---
