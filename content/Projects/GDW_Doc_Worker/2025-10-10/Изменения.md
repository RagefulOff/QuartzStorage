# Плагин `GDW_CleanMD`

## Логирование

С гордостью заявляю - ПОЧИНЕНО, прям полностью (по крайней мере так выглядит).

Насколько я понял, проблема именно в работе с очередью (`_poll_queue`), т.к. очередь была кривой. При выполнении задачи в логи уходили результаты выполнений предыдущей задачи, поэтому в первый запуск логов и не было - потому что предыдущего лога ещё не существует. Прогресс-бар от этого всё равно не воскрес, но логи теперь классные.

> Раньше _poll_queue() иногда не продолжал опрашивать очередь (в зависимости от состояния btn_stop), поэтому сообщения от worker-потоков (лог, progress, done) не доставлялись в GUI — отсюда  отсутствие логов и незавершающаяся работа
> 
> ChatGPT

В самом конце функции, где задаётся передача логов в очередь, было вот так:


## Починка бага с пустым файлом

Если у меня в папке 2 ворд файла: один нормальный, второй пустой, скрипт не обрабатывает тот, который с наполнением, а сразу прекращает работу вообще без ничего, без логов, бес месседжбоксов и т.п..

Происходит это из-за того, что в функции `_start` в блоке подсчёта общего числа сегментов если вдруг в одном из файлов он не найдёт заголовков - скрипт просто остановится. Не будет запущен воркер, не будет передан статус `done`.

Чтобы это поправить, нужно слегка доработать этот блок, добавив обработку исключений try except.

Было:

```python
# Подсчёт общего числа сегментов  
file_segments = {}  
total = 0  
for path in files:  
    segs = self._detect_segments(path, levels)  
    file_segments[path] = segs  
    total += len(segs)  
if total == 0:  
    return messagebox.showinfo("Результат", "Нет заголовков заданного уровня")
```

Стало:

```python
# Подсчёт общего числа сегментов  
file_segments = {}  
total = 0  
for path in files:  
    try:  
        segs = self._detect_segments(path, levels)  
    except Exception as e:  
        self._log(f'Ошибка при анализе {os.path.basename(path)}: {e}')  
        self.errors += 1  
        file_segments[path] = []  
        continue  
    file_segments[path] = segs  
    total += len(segs)  
if total == 0:  
    return messagebox.showinfo("Результат", "Нет заголовков заданного уровня")
```
